Source:

Polygon - A convex polygon made up of vertices
Parameters:
Name Type Attributes Default Description
vertices Array.<Vec2> The vertex points that make up the Polygon. Must result in a convex, nondegenerate shape
mass number "1" The Body's mass
friction number "0.8" The friction coefficient, between 0 and 1 is best
restitution number "0.2" The restitution coefficient, or bounciness, between 0 and 1 is best
options object <optional>
Properties
Name Type Attributes Default Description
isSensor boolean <optional>
 "false" If set to true the Body will not resolve collisions
name string <optional>
 The name of the Body
dampen boolean <optional>
 If set to true, the Body's velocity will be reduced each frame
dampenValue number <optional>
 "0.985" The value that the Body's velocity is reduced by is dampening is true
